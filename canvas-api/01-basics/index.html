The canvas is initially blank. To display something, a script first needs to access the rendering context and draw on it. The element has a method called getContext(), used to obtain the rendering context and its drawing functions. getContext() takes one parameter, the type of context. For 2D graphics, such as those covered by this tutorial, you specify "2d" to get a CanvasRenderingContext2D. হ্যাঁ, `canvas` element থেকে 2D context অবতরণ (obtain) করার পর, `CanvasRenderingContext2D` এর সকল প্রপার্টি এবং মেথড ব্যবহার করে আঁকার কাজ করতে হয়। `getContext('2d')` মেথড দিয়ে 2D context পাওয়া যায়, এবং তারপর `CanvasRenderingContext2D` এর মেথডগুলো যেমন `fillRect()`, `strokeRect()`, `beginPath()`, `moveTo()`, `lineTo()`, `arc()`, `fill()`, `stroke()` ইত্যাদি ব্যবহার করে গ্রাফিক্স ড্রয়িং করা যায়। যেমন: ```javascript // HTML canvas element const canvas = document.getElementById('myCanvas'); const ctx = canvas.getContext('2d'); // 2D context পাওয়া // একেকটি ড্রইং অপারেশন ctx.fillStyle = 'red'; // রঙ নির্বাচন ctx.fillRect(10, 10, 150, 100); // একটি লাল রঙের আয়তক্ষেত্র আঁকা ctx.beginPath(); ctx.arc(200, 75, 50, 0, Math.PI * 2); // একটি বৃত্ত আঁকা ctx.fillStyle = 'blue'; ctx.fill(); ``` এছাড়া `CanvasRenderingContext2D` এর মধ্যে অনেক প্রপার্টি যেমন `lineWidth`, `strokeStyle`, `globalAlpha`, `font` ইত্যাদি রয়েছে, যেগুলি গ্রাফিক্সের স্টাইল এবং প্যামেটার কাস্টমাইজ করতে ব্যবহৃত হয়।
